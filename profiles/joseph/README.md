This directory mirrors the profile owner's home directory.

Replace `.zshrc` and add maintained files such as:

- `.config/ghostty/config`
- `.config/zellij/config.kdl`
- `.config/zellij/layouts/`
- `.config/zellij/plugins/`
- `.config/zellij/scripts/`

Portalgun copies only files present here and backs up replaced destination files.
